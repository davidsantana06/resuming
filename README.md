## üõ†Ô∏è Installation and Execution

For native execution, a **Linux/Unix** machine with **Node.js 22** is required. Configure environment variables using `.env.example` as a template:

- `DATABASE_URL` ‚Äî Database connection string.
- `JWT_SECRET` ‚Äî Secret key for signing JWTs.
- `PORT` ‚Äî Server port.
- `SOURCE_CODE_URL` ‚Äî URL of the source code repository.
- `RELEASES_URL` ‚Äî URL of the releases.

The commands below assume execution from the project root.

Make the script executable:

```bash
chmod +x run.sh
```

Run the application in development or production mode:

```bash
# development mode
./run.sh dev
```

```bash
# production mode
./run.sh prod
```
